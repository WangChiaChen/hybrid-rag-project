ⓘ 只比對名稱完全相同的指標——為什麼？

「手續費淨收益」與「淨手續費收入」這類同義詞目前配不起來：Chroma 預設的 embedding 是英文模型，對中文財務術語沒有鑑別度。

這涉及跨語言 embedding 的語義映射問題，因此我採「高精確度優先」策略——只對詞彙定義完全一致的項目進行對比，避免誤導決策，寧可少配、也不要配錯。

需要語意層面的跨機構比較，請改用「問答與報告」讓 LLM Agent 處理同義詞。

結構化數據採嚴謹比對，語意模糊的術語則交給 AI Agent——這頁展示的正是我們技術的邊界處理方式。
